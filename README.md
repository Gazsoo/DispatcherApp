A real-time dispatcher support system designed to facilitate communication between office dispatchers and field technicians. Built with **Clean Architecture** principles and **CQRS**.

## Technology Stack

* **Backend:** ASP.NET Core Web API, Entity Framework Core, SignalR, MediatR (CQRS)
* **Frontend:** React (SPA), TypeScript, Tailwind CSS
* **Database:** Microsoft SQL Server
* **Testing:** xUnit, Moq

To run the full stack on `localhost:8080` using Docker Compose:

`docker-compose up --build`
